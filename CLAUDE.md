# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Projeto

Landing page do **COC (Centro Preparatório para Olimpíadas do Conhecimento)** — centro de preparação para olimpíadas acadêmicas brasileiras (Matemática, Física, Química, Biologia, História, Robótica, etc.).

O projeto está na **fase inicial de implementação**. Ainda não há framework ou stack definidos — ao sugerir implementações, prefira Next.js/React com Tailwind CSS salvo indicação contrária.

## Assets Disponíveis

- `Logo OFICIAL.webp` — logo oficial do COC (vermelho e verde)
- `paleta.png` — paleta de cores de referência
- `Proposta de Website_...md` — especificação completa das 11 seções do site
- `Extract HTML Design System.md` — template para extração de design system a partir de HTML de referência

## Identidade Visual

- **Cores primárias**: Vermelho e verde (da logo oficial)
- **Tom**: Profissional, motivacional, focado em resultados acadêmicos
- **Elementos visuais da logo**: foguete e órbita atômica

## Estrutura do Site (11 seções)

1. **Hero** — Título impactante, CTAs "Inscreva-se Agora!" e "Explore Nossos Cursos", prova social com selos de medalhas/aprovações
2. **Sobre Nós** — Missão, Visão, Valores (Excelência, Inovação, Dedicação, Ética, Colaboração), histórico
3. **Olimpíadas Abrangidas** — Tabela por olimpíada (OBMEP, OBF, OBQ, OBB, etc.) com links para sites oficiais e CTAs por curso
4. **Nossa Metodologia** — Aulas teóricas, resolução de problemas, simulados, mentoria, material exclusivo, plataforma EAD
5. **Casos de Sucesso** — Depoimentos em vídeo/texto de alunos, pais e professores com fotos e medalhas
6. **Nossa Equipe** — Perfis do corpo docente com fotos profissionais e credenciais
7. **Cursos e Programas** — Catálogo por nível/olimpíada/área, com conteúdo programático, carga horária, formato e preço
8. **Recursos Gratuitos** — Apostilas, provas anteriores, blog, webinars/aulas gravadas
9. **FAQ** — Perguntas frequentes sobre cursos, inscrições e olimpíadas
10. **Contato** — Formulário, telefone/WhatsApp, endereço, redes sociais
11. **Rodapé** — Links rápidos, contato resumido, direitos autorais, política de privacidade

## CTAs Estratégicos

- Hero: "Decole Rumo à Vitória: Inscreva-se Agora!" (vermelho) / "Explore Nossos Cursos" (verde)
- Por olimpíada: "Ver Curso Preparatório para [Olimpíada]"
- Cursos: "Matricule-se Já!" / "Baixar Conteúdo Programático"
- Geral: "Fale Conosco" / "Agende uma Aula Experimental Gratuita"
